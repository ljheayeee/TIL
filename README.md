# Today i Learned

- 좋은 개발자가 되기 위해 하루동안 학습한 내용이나 개발관련 경험들을 기록으로 남긴다

## 작성규칙

- 폴더와 파일명은 영문으로 작성한다.
- 해당 문서를 다시 봤을 때, 추가적인 검색이 없도록 자세히 기록한다.
- 더 많은 공유를 원하는 기록은 블로그에 포스팅한다.
- ref를 명시하고, 원작자가 참고를 허용하는 자료만 사용한다.
